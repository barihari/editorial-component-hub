Accordion

Overview

The Accordion component allows users to expand and collapse content sections, perfect for FAQs or content organization.

Usage

<accordion>
    <accordion-item>
        <h3>Section 1</h3>
        <p>Content here...</p>
    </accordion-item>
</accordion>

Props

Prop Type Default
expanded boolean false

Examples

Content for section 1

Accessibility

The accordion component follows WAI-ARIA practices for accordion patterns.
