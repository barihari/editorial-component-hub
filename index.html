Overview

The Block Quote component is designed to highlight and emphasize quoted content in your documents. It provides various styles and customization options to make your quotes stand out while maintaining readability and visual hierarchy.

Key features include:

  • Multiple styling options
  • Support for attribution/citations
  • Decorative elements (borders, icons)
  • Responsive design
  • Customizable through CSS variables

Installation

Install the blockquote component using your preferred package manager:

npm install @brpress/blockquote
# or
yarn add @brpress/blockquote

Then import the component into your project:

import { BlockQuote } from '@brpress/blockquote';

Usage

Basic usage of the blockquote component:

<BlockQuote
    content="The only way to do great work is to love what you do."
    author="Steve Jobs"
/>

With custom styling:

<BlockQuote
    content="Innovation distinguishes between a leader and a follower."
    author="Steve Jobs"
    variant="styled"
    borderColor="blue-500"
    backgroundColor="gray-100"
/>

Props

Prop Type Default Description
content string required The quote text content
author string undefined Attribution for the quote
variant 'default' | 'styled' | 'decorative' 'default' Visual style variant
className string undefined Additional CSS classes

Styling

The blockquote component can be styled using CSS custom properties:

.blockquote {
    --blockquote-border-color: #1a202c;
    --blockquote-border-width: 4px;
    --blockquote-padding: 1rem;
    --blockquote-font-style: italic;
    --blockquote-text-color: #4a5568;
}

Custom styling example:

.blockquote-elegant {
    --blockquote-border-color: #4299e1;
    --blockquote-background: #f7fafc;
    --blockquote-font-style: normal;
    --blockquote-author-color: #2b6cb0;
}

Accessibility

The blockquote component follows these accessibility best practices:

  • Uses semantic HTML elements (blockquote, cite)
  • Maintains proper heading hierarchy
  • Ensures sufficient color contrast
  • Preserves readability at different zoom levels

Best Practices

  • Use for significant quotes that add value to the content
  • Include proper attribution when available
  • Keep quotes concise and impactful
  • Use appropriate styling for the context
  • Consider mobile responsiveness
